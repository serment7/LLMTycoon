# Inbox

팀 내부 전달·공지·미정리 메모를 모아두는 공간입니다. 작업이 확정되면 정식 문서나 이슈로 옮기세요.

## 용도
- 동료에게 공유할 임시 메모
- 회의에서 나온 결정 초안
- 후속 작업 대기 항목(Todo 리스트)

## 작성 규칙
1. 한 항목당 한 문단. 날짜(YYYY-MM-DD)와 작성자를 앞에 붙입니다.
2. 확정된 사항은 `docs/` 하위 문서로 승격하고, 이 파일에서는 해당 항목을 제거합니다.
3. 1주일 이상 방치된 항목은 아카이브 또는 폐기합니다.

## 항목 템플릿
```
- [YYYY-MM-DD / 작성자] 주제: 내용 요약 (후속 조치: ...)
```
